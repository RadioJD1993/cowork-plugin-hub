# Standard Example Plugin

Demonstrates a skill, a command, and a GitHub MCP connector working together.

| Component | File |
|-----------|------|
| Manifest | `.claude-plugin/plugin.json` |
| Connector config | `.mcp.json` |
| Connector setup | `CONNECTORS.md` |
| Auto-firing skill | `skills/summarize/SKILL.md` |
| Slash command | `commands/fetch-and-summarize.md` |

## Install

```bash
claude plugin install standard-example@cowork-plugin-hub
```
