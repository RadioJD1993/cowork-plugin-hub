# CoWork Plugin Catalog

A public, static Hugging Face Space for browsing the sanitized CoWork Plugin Hub catalog.

Update this folder by running `npm run catalog` from the repository root, then publish the folder contents to a Hugging Face Space. The Space serves `index.html`, `catalog.json`, and `agents.md` only; it does not contain private plugin bundles or local configuration.
